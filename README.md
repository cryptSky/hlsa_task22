## Features
* Shortens url
* Copy shortened url to clipboard
* Next.js + Mongodb connection
* Ssr with react

## Prerequisite
To build this project, you require:
- Visual studio code or any other code editor
- Node installed
- Basic knowledge of Typescript
- Basic knowledge of React
- ES6
